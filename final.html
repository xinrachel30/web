Jeopardy Parody

Shreya Daggolu and Xinqing Lin, Period 5

  • Project Description

    • This is a parody of the classic Jeopardy game. It is a single-player game where you can test your knowledge of high school and pop culture.

  • Instructions

    1. Open and run the Python program.
    2. Using the board below, type the desired topic and value into the Thonny shell.
    3. Type your response into the Thonny shell. Upon entering, the program will let you know if you were correct or not and your updated score.
    4. Repeat until you are finished with the game. Once done, you will be able to see a line graph depicting your progress.

  • How it works

    • The questions and answers are all in a txt file, organized by topic and value. The organize function will organize all of the values of the txt file into a dictionary of dictionaries. This makes it easier to refer to a particular tile. The score function takes the topic, value, and response. It will compare the response to the actual answer using the dictionary of dictionaries, topic, and value. If the response matches the answer, it will print a success message. Otherwise, it will print a failure message and the correct answer. The start_game function uses a while loop to continue the game until all of the questions are answered. The function ultimately produces a list This list will be used to create a line graph so that the player can see their progress throughout the game.

  • Bugs, Errors and Things to be Aware of

    • The topic, value, and response typed into the Thonny shell has to be the exact spelling and capitalization. If not, the entire game will start over.
    • Technically, you can repeat questions until you reach the 30, the total number of existing questions.
    • The dictionary of dictionaries produced is ordered weirdly. The question and answer corresponding to the value 1000 come before the ones for the value 200. However, this doesn't interfere with gameplay.
